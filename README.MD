### Configuration (`about:config`)

**Required**
- `toolkit.legacyUserProfileCustomizations.stylesheets = true`
- `svg.context-properties.content.enabled = true`

**Windows**
- `widget.windows.mica = true`
- `widget.windows.mica.toplevel-backdrop = 2`
